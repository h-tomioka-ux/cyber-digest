# 🛡️ サイバーセキュリティ・ダイジェスト 2026-05-01

> ソース: RSS 6フィード (12記事) ／ NotebookLM分析

## 🔴 重大インシデント

*   **Mini Shai-Hulud サプライチェーン攻撃による大規模な認証情報窃取**
    PyPIのLightningやNPMのintercom-clientなど、累計ダウンロード数が数千万回に及ぶパッケージが侵害され、**1,800人以上の開発者が被害を受けました** [1-4]。TeamPCPによるこの攻撃は、感染したマシンからAWSキー、GitHub/NPMトークン、SSHキー、クラウド認証情報などの機密データを盗み出し、GitHubリポジトリへ公開することを目的としています [2, 5-7]。
*   **Checkmarxの盗まれたデータがダークウェブに公開**
    Checkmarxは、GitHubリポジトリから盗まれたデータがサイバー犯罪グループ**LAPSUS$によってダークウェブ上に公開された**ことを確認しました [8]。調査の結果、Trivyスキャナーを標的としたTeamPCPによる攻撃が、GitHubリポジトリへの不正アクセスのきっかけとなった可能性が高いとされています [8]。
*   **60万件以上のRobloxアカウントをハッキングしたグループの摘発**
    ウクライナ当局は、**61万件以上のRobloxゲーミングアカウントをハッキング**し、ロシアのウェブサイトで販売して約22万5,000ドルの利益を得ていた3人のグループを逮捕しました [9]。主犯格は19歳の居住者で、昨年からゲーミングフォーラムで知り合った仲間と共にアカウントへの不正アクセスを繰り返していました [9]。

## 🟠 脆弱性情報

*   **Microsoft SharePoint Server のなりすましの脆弱性**
    *   **CVSSスコア:** ソースに記載なし
    *   **概要:** **CVE-2026-32201**。認証不要でネットワーク経由のなりすましが行われる可能性があり、マイクロソフトは既に**悪用が確認されている**と公表しています [10]。
    *   **推奨対応:** Microsoft UpdateやWindows Updateを用いて、2026年4月のセキュリティ更新プログラムを早期に適用してください [11]。
*   **Adobe Acrobat および Reader の脆弱性（APSB26-44）**
    *   **CVSSスコア:** ソースに記載なし
    *   **概要:** PDF作成・閲覧ソフトウェアにおいて、ユーザーが悪意のあるコンテンツを開いた場合に**任意のコードが実行される**などの恐れがあります [12]。
    *   **推奨対応:** 最新のセキュリティバージョン（例：Acrobat DC Continuous 26.001.21431など）へ更新してください [13]。
*   **Cisco ASA および FTD における複数の脆弱性**
    *   **CVSSスコア:** ソースに記載なし
    *   **概要:** **CVE-2025-20333**（認証後の任意コード実行）と**CVE-2025-20362**（未認証でのURLアクセス）を組み合わせた攻撃が確認されています [14]。侵害されると「FIRESTARTER」バックドアが設置され、**修正パッチ適用後も永続化する**特性があります [15, 16]。
    *   **推奨対応:** 修正済みバージョンへのアップデートに加え、侵害の痕跡調査、パスワードや鍵のリセット、必要に応じた機器の再イメージ化を検討してください [16-18]。
*   **OpenEMRにおける38件の脆弱性**
    *   **CVSSスコア:** ソースに記載なし（「Critical」を含む）
    *   **概要:** オープンソースの電子医療記録プラットフォームにおいて、SQLインジェクションやパストラバーサルなど38件の脆弱性が発見されました [19]。これらは数億人の患者データへの不正アクセスや、サーバー上でのリモートコード実行につながる可能性があります [19]。
    *   **推奨対応:** プラットフォームのパッチが公開されているため、速やかに適用してください [19]。

## 🟡 攻撃キャンペーン

*   **DEEP#DOOR Pythonバックドアによる認証情報窃取**
    フィッシング経由で配布されるステルス性の高いPythonベースのバックドアフレームワークが報告されました [20, 21]。Rustベースのトンネリングサービス「bore[.]pub」を悪用してC2通信を行い、ブラウザ、SSH、AWS/Azure等のクラウド認証情報を広範囲に収集します [20, 22, 23]。
*   **SMSブラスターを用いたフィッシング詐欺**
    カナダ当局は、偽の携帯電話基地局（SMSブラスター）を使用して周囲のスマートフォンにフィッシング詐欺メールを送信していた3人の男を逮捕しました [24]。このデバイスは数万台のデバイスを騙して接続させ、銀行の認証情報などを盗む偽サイトへ誘導していました [24]。
*   **PhantomRPC による特権昇格攻撃**
    WindowsのRPCアーキテクチャの弱点を突く**PhantomRPC**と呼ばれる手法が特定されました [19]。攻撃者は限定的なローカルアクセスから、特定のRPCインターフェースを模倣することで、権限をSYSTEMレベルに昇格させることが可能です [19]。

## 🤖 AIセキュリティ

*   **AIによるサイバー犯罪の「産業化」と攻撃時間の短縮**
    攻撃者が**WormGPT、FraudGPT、HexStrike AI**などのAIツールを「フォースマルチプライヤー」として活用し、高度な攻撃を機械的なスピードで実行しています [25-27]。脆弱性の公開から悪用開始までの時間は、かつての数日から**数時間単位にまで短縮**されています [28]。
*   **Anthropicによる防御用AI「Claude Security」の公開**
    AIによる攻撃の高速化に対抗するため、Anthropicは企業向けに**Claude Security**のパブリックベータ版を公開しました [29]。リポジトリをスキャンして脆弱性を特定し、信頼性の高い評価と修正用パッチの生成を支援することで、発見から修正までの時間を大幅に削減します [30, 31]。
*   **AI開発環境を標的としたワーム型攻撃**
    PyTorch Lightningパッケージの侵害では、盗まれたGitHubトークンを使用して、**Anthropicの「Claude Code」を装った**悪意のあるコミットを複数のブランチに注入するワームのような拡散活動が確認されました [32]。

*   **Exchange OnlineにおけるレガシーTLSの完全廃止**: Microsoftは、POPおよびIMAP接続でのTLS 1.0および1.1のサポートを**2026年7月から完全に停止**する計画を発表しました [33]。
*   **ブラウザ拡張機能によるユーザーデータの合法的な販売**: 調査により、80個以上の拡張機能がプライバシーポリシーに明記した上で、数百万人のユーザー履歴を収集・販売している実態が判明しました [24]。
*   **アーカイブ（arXiv）のプレプリントによる機密データ漏洩**: arXivに投稿された270万件の論文ファイルを分析した結果、GitリポジトリやAPIキー、秘密鍵などの不要なデータがLaTeXソース等に含まれたまま公開されている事例が多数発見されました [9]。
