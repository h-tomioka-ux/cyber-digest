# 🛡️ サイバーセキュリティ・ダイジェスト 2026-07-14

> ソース: RSS 6フィード (12記事) ／ NotebookLM分析

## 🔴 重大インシデント

*   **Fortinet製品の認証情報大規模漏洩（FortiBleed）**
    194カ国にわたる企業や政府機関の**86,644台以上の機器のログイン情報**が、攻撃者のデータベースから発見されました [1]。漏洩した情報には国内企業に関連するものも含まれており、VPNへの不正アクセスから内部ネットワークへの侵入やWindowsドメイン資格情報の窃取に繋がる恐れがあります [2]。
*   **Progress ShareFileにおける「信頼できる外部の脅威」**
    Progress社は、**Windows版Storage Zone Controller**を運用する顧客に対し、サーバーのシャットダウンを緊急要請しました [3]。同社は慎重を期して対象アカウントへのアクセスを一時的に停止しており、内部および外部の専門家と協力して脅威の特定にあたっています [3]。
*   **Citrix Bleed 2の悪用によるDragonForceランサムウェア攻撃**
    攻撃者が脆弱性（CVE-2025-5777）を悪用し、**DragonForceランサムウェア**を配備する事例が確認されました [4]。攻撃者はシステム権限への昇格や正規のリモート管理ツールの設置など、標準化されたプレイブックに基づいた一連の攻撃活動を行っています [4]。
*   **Centers Laboratoryにおけるデータ漏洩**
    医療機関向け検査サービス等を提供するCenters Laboratoryにてデータ漏洩が発生し、**約54万人の個人に影響**が及んでいます [5]。

## 🟠 脆弱性情報

*   **RabbitMQ OAuthシークレット漏洩の脆弱性**
    *   **CVE番号:** CVE-2026-5721 [6]
    *   **CVSSスコア:** **8.7** [6]
    *   **概要:** 管理Webインターフェースの古いエンドポイントから、認証なしでOAuthクライアントシークレットを取得できる可能性があります [6, 7]。これにより、攻撃者がブローカーになりすまして管理者トークンを取得し、設定やメッセージを完全に制御される恐れがあります [7]。
    *   **推奨対応:** RabbitMQを修正済みの最新バージョン（4.3.0、4.2.6など）へ直ちに更新し、OAuthクライアントシークレットをローテーションしてください [8, 9]。
*   **Microsoft Exchange Server のなりすましの脆弱性**
    *   **CVE番号:** CVE-2026-42897 [10]
    *   **CVSSスコア:** ソースに記載なし
    *   **概要:** 既に**悪用が確認されている**Exchange Serverの脆弱性で、なりすましが行われる可能性があります [10]。
    *   **推奨対応:** Microsoft Update等を利用し、2026年6月のセキュリティ更新プログラムを早期に適用してください [10, 11]。
*   **Adobe AcrobatおよびReaderにおける任意のコード実行の脆弱性**
    *   **識別番号:** APSB26-63 [12]
    *   **CVSSスコア:** ソースに記載なし
    *   **概要:** 脆弱性を悪用したコンテンツをユーザーが開いた場合、**任意のコードが実行される**などの可能性があります [12]。
    *   **推奨対応:** 製品のメニューからアップデートをチェックするか、最新バージョン（26.001.21662等）をダウンロードして適用してください [13]。
*   **Zimbra Classic Web Clientの重大な脆弱性**
    *   **CVE番号:** 未割り当て [14]
    *   **CVSSスコア:** ソースに記載なし
    *   **概要:** 蓄積型クロスサイトスクリプティング（XSS）により、細工されたメールを開くだけで任意のコードが実行されるリスクがあります [14]。
    *   **推奨対応:** Zimbraが提供する最新のアップデートを適用してください [14]。

## 🟡 攻撃キャンペーン

*   **CrashStealer macOSマルウェア**
    **Appleによって公証（Notarized）されたドロップアプリ**を使用し、macOSのセキュリティ機能「Gatekeeper」を回避して感染を広げる情報窃取マルウェアです [15, 16]。ブラウザの認証情報、80種類以上の暗号資産ウォレット、14種類のパスワードマネージャーのデータを標的としています [17, 18]。
*   **SHELLSTORM：大規模なWebシェルアクセスブローカー活動**
    WordPressプラグインの27個の脆弱性を悪用し、**140万以上のドメインを標的**にWebシェルを配置する大規模なキャンペーンです [14]。中国系の攻撃者による活動と推定されており、バックドアの配備にも利用されています [14]。
*   **ModHeaderブラウザ拡張機能への悪意あるコード混入**
    160万以上のインストール数を誇る人気拡張機能に、**閲覧履歴を収集する休眠状態のコード**が含まれていたことが判明しました [19]。既にストアから削除されていますが、利用者は手動での削除と、保存していたシークレット情報のローテーションが推奨されています [19, 20]。

## 🤖 AIセキュリティ

*   **HalluSquatting攻撃**
    AIコーディングアシスタントが生成する**「ハルシネーション（幻覚）」を利用**した新しい攻撃手法です [14]。AIが実在しないリソース名を提案することを利用し、攻撃者がその名前で悪意あるパッケージを先回りして登録しておくことで、開発者にボットネットをインストールさせます [14]。
*   **AIエージェントの悪用と防御回避**
    悪意あるコードを検出するために構築されたAIエージェントが、逆にそのコードを実行するように騙される「Friendly Fire」という現象が報告されています [21, 22]。また、プロンプトインジェクションを用いてAIエージェントから資格情報を漏洩させたり、安全制御をバイパスしたりする実証実験も行われています [14, 23]。
*   **AIインフラを標的とした暗号資産マイニング**
    LiteLLM Proxyなどの**AIゲートウェイが侵害**され、Amazon Bedrockなどのサービスに接続された計算リソースが、許可なく暗号資産のマイニングに悪用される事例が確認されました [4]。
