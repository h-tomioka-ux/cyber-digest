# 🛡️ サイバーセキュリティ・ダイジェスト 2026-07-16

> ソース: RSS 6フィード (12記事) ／ NotebookLM分析

2026年7月16日に取得された情報を基に、サイバーセキュリティニュースの分析結果を報告します。

## 🔴 重大インシデント
*   **FortiBleed：Fortinet製品の大規模な認証情報漏えい**
    194カ国にわたる企業や政府機関に属する86,644台以上のFortiGate機器のログイン情報が、攻撃者の運用するデータベースから流出したことが確認されました [1]。この事案は過去のインシデントで取得された情報の再利用が主因とされていますが、現在も有効な認証情報が含まれている可能性や、設定ファイルの解析により管理者パスワードが解読されるリスクが指摘されています [1, 2]。漏えいした情報が悪用された場合、内部ネットワークへの侵入やActive Directoryの資格情報窃取にまで被害が拡大する恐れがあります [3]。

## 🟠 脆弱性情報
*   **Microsoft SharePoint Server の特権の昇格の脆弱性 (CVE-2026-56164)**
    *   **CVSSスコア：** 記載なし（ただし、悪用が公表されているゼロデイ脆弱性） [4, 5]
    *   **概要：** 認証なしでリモートから特権を昇格させることが可能な脆弱性です [5]。
    *   **推奨対応：** Microsoft Update または Windows Update を使用して、2026年7月のセキュリティ更新プログラムを直ちに適用してください [6]。
*   **Adobe ColdFusion におけるパス・トラバーサルの脆弱性 (CVE-2026-48318)**
    *   **CVSSスコア：** 9.9 [7]
    *   **概要：** 任意のコード実行につながる恐れのある深刻なパス・トラバーサルの脆弱性です [7]。
    *   **推奨対応：** ColdFusion 2025 Update 11 または ColdFusion 2023 Update 22 に更新してください [8]。
*   **VMware Avi Load Balancer における認証バイパスの脆弱性 (CVE-2026-47865)**
    *   **CVSSスコア：** 9.8 [9]
    *   **概要：** ネットワークアクセスを持つ攻撃者が Avi コントロールプレーンにアクセスできる可能性があります [9]。
    *   **推奨対応：** Broadcom から提供されている修正プログラムを適用してください [9]。

## 🟡 攻撃キャンペーン
*   **OkoBot：ハードウェアウォレットを狙うシードフレーズ窃取**
    2025年4月から活動しているマルウェアフレームワークで、LedgerやTrezorのデスクトップアプリに悪意のあるページを注入します [10, 11]。デバイスの接続を監視し、アプリ内部に偽のリカバリーページを表示してユーザーにシードフレーズを入力させる巧妙な手口を用いています [11, 12]。
*   **Windowsの「バインドリンク」機能を悪用したEDR回避攻撃**
    Windowsの正規機能である「バインドリンク」を悪用してファイルパスをリダイレクトし、EDR製品の監視を回避する手法が報告されました [13]。攻撃者はこの機能を用いて悪意のあるDLLを信頼されたパスとしてロードさせたり、マルウェアの実行を隠蔽したりすることが可能です [14, 15]。

## 🤖 AIセキュリティ
*   **TuxBot v3 Evolution：LLMによるボットネット開発の加速**
    IoTボットネット「TuxBot」の開発に大規模言語モデル（LLM）が利用された形跡が確認されました [16]。コード内にはLLMの思考プロセスや、AIが生成した際に付与される安全性の免責事項が残されており、開発者がAIの助けを借りて多機能な攻撃ツールを構築している実態が明らかになりました [16, 17]。
*   **AI開発環境「Cursor」における未修正のコード実行脆弱性**
    AI支援型エディタである「Cursor」において、リポジトリのルートに配置された悪意のある「git.exe」を自動実行してしまう脆弱性が報告されました [18, 19]。7ヶ月以上前に報告されたものの未修正の状態が続いており、AIツールの普及に乗じた新たな脅威となっています [20, 21]。
