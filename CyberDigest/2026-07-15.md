# 🛡️ サイバーセキュリティ・ダイジェスト 2026-07-15

> ソース: RSS 6フィード (12記事) ／ NotebookLM分析

2026年7月14日から15日にかけて報告されたサイバーセキュリティニュースに基づき、ご指定の形式で分析結果を報告します。

## 🔴 重大インシデント
*   **スペイン警察による1億4,000万ユーロ規模のサイバー詐欺グループ摘発**
    スペイン警察が、投資詐欺やビジネスメール詐欺（BEC）に関与した大規模な犯罪組織を解体し、4人を逮捕しました [1, 2]。この組織は800以上の銀行口座を管理し、マネーミュール（資金移動役）を利用して、盗み出した計1億4,000万ユーロ（約1億6,000万ドル）もの資金を洗浄していました [2, 3]。
*   **SynopsysおよびBoschに対するデータ侵害の主張（真偽不明）**
    D1Rという新たなランサムウェアグループが、Synopsysのサイトの脆弱性を悪用して4万件の顧客データベースにアクセスし、それを利用してBoschから知的財産を盗んだと主張しています [4, 5]。しかし、Synopsys側は調査の結果、侵害の証拠は見つかっておらず主張は根拠のないものだと否定しています [6]。

## 🟠 脆弱性情報
*   **Microsoft 2026年7月 セキュリティ更新プログラム**
    *   **CVSSスコア:** 最大 9.9（CVE-2026-57092 - Windows VMSwitch RCE）/ 9.8（CVE-2026-50522 - SharePoint RCE）[7-9]
    *   **概要:** 過去最多となる622件の脆弱性を修正しました [10]。うち2件（SharePointのCVE-2026-56164とAD FSのCVE-2026-56155）は、攻撃者に既に悪用されているゼロデイ脆弱性です [10, 11]。
    *   **推奨対応:** 早急にWindows Update等を用いて最新のパッチを適用してください [10, 12]。
*   **SAP NetWeaver Application Server ABAP の脆弱性**
    *   **CVSSスコア:** 9.9（CVE-2026-44747）[13]
    *   **概要:** メモリ管理の論理エラーに起因する境界外書き込みの欠陥で、認証された攻撃者がデータへの不正アクセス、改ざん、またはシステム停止を引き起こす可能性があります [13]。
    *   **推奨対応:** 修正済みのABAPカーネルバージョンをインストールしてください。暫定的な回避策として、特定のICFノードの無効化も提案されています [14]。
*   **Adobe ColdFusion のクリティカルな脆弱性**
    *   **CVSSスコア:** クリティカル [15, 16]
    *   **概要:** 任意のコード実行や権限昇格につながる8件のクリティカルな欠陥を含む、計13件の脆弱性が修正されました [16]。
    *   **推奨対応:** ColdFusion 2025 update 11 または ColdFusion 2023 update 22 へ早急に更新してください [17]。

## 🟡 攻撃キャンペーン
*   **FortiBleed（Fortinet製品の認証情報漏えい）**
    世界194カ国の企業や政府機関に属する86,644台以上のFortiGate等の機器のログイン情報が漏えいしたとされる事案です [18]。過去のインシデントで取得された情報の再利用が主因とされていますが、現在も有効な認証情報が含まれている可能性が指摘されており、国内企業の情報も含まれていることが確認されています [18, 19]。

## 🤖 AIセキュリティ
*   **Claude for Chrome の「Confused Deputy」脆弱性**
    Anthropic社のブラウザ拡張機能「Claude for Chrome」において、他の悪意ある拡張機能がスクリプトを実行し、ユーザーのGmail、Googleドキュメント、カレンダーを読み取らせることができる脆弱性が報告されました [20, 21]。ユーザーが「確認なしで実行」設定にしている場合、攻撃者はサイレントにデータを窃取することが可能です [22, 23]。
*   **AIを活用した脆弱性発見の加速**
    Microsoftは、AIエージェント型スキャンシステム「MDASH」などの導入により、脆弱性発見のスピードが飛躍的に向上していると発表しました [24, 25]。今回の622件という膨大なパッチ件数の一因も、AIによるコード解析の効率化にあるとしています [24]。
*   **HalluSquatting攻撃**
    AIのハルシネーション（幻覚）を悪用し、存在しないソフトウェアパッケージを推奨させてマルウェアをインストールさせる「HalluSquatting」攻撃の手法が指摘されています [26, 27]。

## 🔵 業界動向
*   **脆弱性管理の優先順位付けの変化:** パッチ件数の膨大化に伴い、CVSSスコアのみに頼るのではなく、KEV（既知の悪用された脆弱性）カタログやEPSSなどの実悪用指標に基づいた管理へのシフトが求められています [28, 29]。
*   **AIガバナンスと監査の必要性:** AI生成コードの普及に伴い、CISOにはAIツールの使用を統治し、本番環境に到達する前にソフトウェアリスクを特定する新しい監査戦略が求められています [30, 31]。
*   **ビジネス整合型リスク管理への移行:** 技術的なデータから、ビジネスへの影響や継続的なリスクサイクルを重視するリスク管理モデルへの移行が推奨されています [32, 33]。

## 🇯🇵 国内情報
*   **JPCERT/CCによる一連の注意喚起:** 2026年6月および7月のMicrosoft製品、Adobe Acrobat/Reader、およびFortinet製品に関するセキュリティ更新と脆弱性について、注意喚起が行われています [34-36]。
*   **FortiBleedに関する国内への影響:** JPCERT/CCは「FortiBleed」事案において国内組織の情報が含まれていることを確認しており、VPN機器への不正アクセスを起点とした内部ネットワーク侵入を防ぐため、管理者パスワードの変更、多要素認証の有効化、Active Directoryのログ調査を強く推奨しています [19]。
