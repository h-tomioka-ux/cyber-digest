# 🛡️ サイバーセキュリティ・ダイジェスト 2026-04-27

今日（2026-04-27）取得した最新のサイバーセキュリティニュースに基づき、以下の通り分析・報告します。

## 🔴 重大インシデント
*   **医療機器大手Strykerへの大規模なデータ消去攻撃**
    イランに関連するとみられるハッカー集団「Handala」が、医療機器大手Strykerに対し大規模なデータ消去（ワイパー）攻撃を仕掛けたと主張しました [1, 2]。同社は世界79カ国の拠点が閉鎖に追い込まれ、20万台以上のシステムやデバイスに影響が出たと報じられています [3, 4]。
*   **仮想通貨取引所Grinexに対する1,374万ドル規模のハッキング**
    キルギスの仮想通貨取引所Grinexが、大規模なサイバー攻撃により約1,374万ドル（約10億ルーブル）相当のユーザー資金を盗難されたとして業務を停止しました [5]。同社はこの攻撃について、国家機関による高度なリソースが投入されたものであると述べています [5]。
*   **中国に関連するAPT「GopherWhisper」による政府機関攻撃**
    モンゴル政府機関を標的とした、中国拠点の新たなAPTグループ「GopherWhisper」の活動が明らかになりました [6, 7]。このグループは、SlackやDiscordなどの正当なサービスをC&C通信に悪用する複数のGo言語ベースのバックドアを駆使しています [7-9]。

## 🟠 脆弱性情報
*   **Microsoft SharePoint Server のなりすましの脆弱性 (CVE-2026-32201)**
    *   **CVSSスコア:** ソースに具体的な数値の記載なし（悪用が確認されているゼロデイ） [10, 11]
    *   **概要:** 認証不要でネットワーク経由でのなりすましや、信頼されたコンテンツの偽装を許す可能性があります [10, 12]。
    *   **推奨対応:** Microsoft Update等を通じてセキュリティ更新プログラムを速やかに適用してください [13]。
*   **Apache ActiveMQ Classic のコードインジェクションの脆弱性 (CVE-2026-34197)**
    *   **CVSSスコア:** 8.8 [14]
    *   **概要:** 入力検証の不備により、リモートから任意のコードを実行される恐れがあり、CISAの既知の悪用された脆弱性カタログ（KEV）に追加されました [14, 15]。
    *   **推奨対応:** 2026年4月30日までに修正プログラムを適用してください [14]。
*   **Adobe AcrobatおよびReaderの脆弱性 (APSB26-43)**
    *   **CVSSスコア:** ソースに具体的な数値の記載なし [16]
    *   **概要:** 悪用されると任意のコードを実行される可能性があり、アドビは本脆弱性の悪用をすでに確認しています [11, 16]。
    *   **推奨対応:** メニューの「ヘルプ」から「アップデートの有無をチェック」をクリックし、最新版に更新してください [17]。

## 🟡 攻撃キャンペーン
*   **ロシア系グループ「Forest Blizzard」によるルーター侵害キャンペーン**
    ロシア軍参謀本部情報総局（GRU）に関連する「Forest Blizzard」が、古いTP-LinkやMikroTikルーターの既知の脆弱性を突き、Microsoft Officeの認証トークンを大規模に窃取しています [18, 19]。マルウェアをインストールせず、DNS設定を変更してトラフィックを傍受する手法が特徴です [18, 20]。
*   **「Starkiller」フィッシング・アズ・ア・サービス**
    正規のログインページを動的にロードし、リレーとして機能する高度なフィッシング基盤が確認されました [21, 22]。ユーザーの入力をリアルタイムで正規サイトへ転送することで、多要素認証（MFA）を無効化する能力を持ちます [22, 23]。
