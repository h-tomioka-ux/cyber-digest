# 🛡️ サイバーセキュリティ・ダイジェスト 2026-05-18

> ソース: RSS 6フィード (12記事) ／ NotebookLM分析

今日（2026-05-18）取得したサイバーセキュリティニュースに基づき、以下の通り分析結果をまとめます。

## 🔴 重大インシデント

*   **7-Elevenでのデータ漏洩の確認とShinyHuntersによる身代金要求**
    世界最大のコンビニエンスストアチェーンである7-Elevenは、ハッカー集団「ShinyHunters」によるシステム侵入とデータ窃取を認めました [1]。ハッカー側は、フランチャイズ申請時の個人情報や企業データを含む60万件以上のSalesforceレコードを盗んだと主張しており、25万ドルでのデータ販売を提示しています [2, 3]。
*   **Cisco製ファイアウォールを標的とした持続的な攻撃（ArcaneDoor）**
    CiscoのASAおよびFTD機器において、脆弱性を悪用して「FIRESTARTER」と呼ばれるバックドアを設置し、永続的なアクセスを確保する攻撃キャンペーンが続いています [4, 5]。この攻撃は「ArcaneDoor」の背後にいる脅威アクターによるもので、一般的な監視手法では発見が困難な特性を持っています [5]。

## 🟠 脆弱性情報

*   **MiniPlasma (Windows 0-Day)**
    *   **CVSSスコア:** 7.0（2020年当時のCVE-2020-17103としての評価）[6]
    *   **概要:** Windows Cloud Files Mini Filter Driver (cldflt.sys) に存在する特権昇格の脆弱性で、最新の2026年5月パッチ適用済みシステムでもSYSTEM権限を奪取可能です [7, 8]。
    *   **推奨対応:** 現時点でMicrosoftからの公式な再修正は未発表ですが、Insider Preview Canary版では動作しないとの報告があります [9]。
*   **DirtyDecrypt / DirtyCBC (Linux Kernel)**
    *   **CVSSスコア:** 記載なし（CVE-2026-31635に関連）[10]
    *   **概要:** Linuxカーネルのrxgkモジュールにおけるページキャッシュ書き込みの不備により、ローカルでroot権限を取得可能です [10, 11]。
    *   **推奨対応:** 最新のカーネルアップデートを速やかに適用すること、または特定の緩和策（IPsec VPN等に影響あり）の実施が推奨されます [12, 13]。
*   **Ivanti Xtraction の重大な脆弱性 (CVE-2026-8043)**
    *   **CVSSスコア:** 9.6 [14]
    *   **概要:** 認証された攻撃者が機密ファイルを読み取ったり、任意のHTMLファイルを書き込んだりすることが可能で、情報漏洩やクライアントサイド攻撃につながります [15]。
    *   **推奨対応:** バージョン 2026.2 以降へのアップデートを適用してください [15]。
*   **Fortinet 製品のコード実行の脆弱性 (CVE-2026-44277, CVE-2026-26083)**
    *   **CVSSスコア:** 9.1 [15]
    *   **概要:** FortiAuthenticator や FortiSandbox のWeb UIにおいて、未認証の攻撃者が任意のコードやコマンドを実行できる可能性があります [15]。
    *   **推奨対応:** 修正済みバージョン（FortiAuthenticator 6.5.7/6.6.9/8.0.3等）へのアップデートが推奨されます [15]。
*   **SAP S/4HANA および Commerce Cloud の脆弱性 (CVE-2026-34260, CVE-2026-34263)**
    *   **CVSSスコア:** 9.6 [16]
    *   **概要:** SQLインジェクションや認証チェックの欠如により、任意のサーバーサイドコード実行や機密データの漏洩が発生する恐れがあります [16, 17]。
    *   **推奨対応:** SAPが公開しているセキュリティパッチを適用してください [16]。

## 🟡 攻撃キャンペーン

*   **開発者環境を標的とした資格情報窃取（TeamPCP / Shai-Hulud）**
    npm、PyPI、Docker Hubなどのパッケージマネージャーを介して、開発者のワークステーションからAPIキー、クラウド資格情報、SSHキーなどを盗み出すキャンペーンが活発化しています [18, 19]。特に「Shai-Hulud」ワームは、感染した開発環境を資格情報の収集ポイントとして悪用し、数千の機密情報を流出させています [19]。
*   **Cisco Firepower デバイスを狙う UAT-4356**
    「ArcaneDoor」として知られるキャンペーンでは、2025年9月頃から修正パッチ公開前に侵害が始まっていた可能性が指摘されており、現在もバックドアの設置による永続化が狙われています [5]。

## 🤖 AIセキュリティ

*   **OpenClaw AIアシスタントの脆弱性チェーン「Claw Chain」**
    AIアシスタント「OpenClaw」において、4つの脆弱性（CVE-2026-44112など、最大CVSS 9.6）を連鎖させることで、サンドボックスを脱出し、ホストシステムにバックドアを設置することが可能です [20, 21]。プロンプトインジェクションや悪意のあるプラグインを通じて攻撃がトリガーされるリスクがあります [22]。
*   **AIコーディングアシスタントによるサプライチェーンリスク**
    AIエージェントやコーディングアシスタントが、機密データを含むローカルの開発コンテキストを処理する過程で、プロンプトやログに秘密情報が混入し、それが外部に流出する新たな攻撃局面が懸念されています [23]。

## 🔵 業界動向

*   **開発者ワークステーションの「サプライチェーン境界」化**
    ソフトウェアの提供プロセスがGitにプッシュされる前から始まっていることを踏まえ、開発者のPCをサプライチェーンの重要な境界として保護すべきという認識が広まっています [24, 25]。
*   **攻撃の高速化と自動化**
    AIを活用したツールの登場により、脆弱性の発見から悪用までの時間が数秒単位にまで短縮されており、従来の人間によるレビューが追いつかない現状があります [26, 27]。
*   **Windows 11 の UI カスタマイズ機能の再導入**
    Microsoftは、ユーザーからの要望が多かったタスクバーの移動（上下左右）やサイズ調整、スタートメニューの推奨コンテンツ表示設定などの変更をInsider Preview版でテストしています [28, 29]。

    JPCERT/CCは、国内でも依然として脆弱なCisco製ホストが多数存在することを確認しており、修正済みであっても侵害が潜伏している可能性があるとして調査を推奨しています [33]。
*   **GUARDIANWALL MailSuite の脆弱性 (CVE-2026-32661) と悪用確認**
    キヤノンマーケティングジャパンの「GUARDIANWALL MailSuite」において、未認証で任意のコード実行が可能な脆弱性が公表されました [34]。オンプレミス版において、既にこの脆弱性を悪用した攻撃が確認されており、極めて警戒が必要です [34]。
