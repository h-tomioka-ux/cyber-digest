# 🛡️ サイバーセキュリティ・ダイジェスト 2026-05-08

> ソース: RSS 6フィード (12記事) ／ NotebookLM分析

2026年5月8日時点のソースに基づき、サイバーセキュリティニュースを以下の通り分析します。

## 🔴 重大インシデント
*   **Instructure（Canvas）への大規模な脅迫キャンペーン（ShinyHunters）**
    脅迫集団「ShinyHunters」が、教育技術大手InstructureのCanvasログインポータルを改ざんし、約330の教育機関が影響を受けました [1], [2]。この集団は、2億8000万人分の学生および職員のデータを盗み出したと主張しており、5月12日までに身代金を支払わなければデータを漏洩させると脅迫しています [3], [4]。
*   **Palo Alto Networksのファイアウォールにおけるゼロデイ脆弱性の悪用（国家関与の疑い）**
    Palo Alto NetworksのPAシリーズおよびVMシリーズのファイアウォール（User-ID認証ポータル）に存在するゼロデイ脆弱性（CVE-2026-0300）が、国家支援を受けたとみられる攻撃グループ「CL-STA-1132」によって悪用されました [5], [6]。攻撃者はroot権限でのリモートコード実行を行い、Active Directoryの情報を標的にしたほか、痕跡消去を徹底するなど高度な技術を用いています [5], [7]。

## 🟠 脆弱性情報
*   **Ivanti Endpoint Manager Mobile (EPMM) の脆弱性**
    *   **CVE番号 / CVSSスコア:** CVE-2026-6973 / 7.2 (High) [8]
    *   **概要:** 管理者権限を持つ認証済みのユーザーが、不適切な入力検証を悪用してリモートでコードを実行できるゼロデイ脆弱性です [8], [9]。
    *   **推奨対応:** EPMMのバージョン 12.6.1.1、12.7.0.1、12.8.0.1 以降への更新が推奨されています [8]。
*   **Adobe AcrobatおよびReaderの脆弱性（APSB26-44）**
    *   **CVSSスコア:** ソースに記載なし（JPCERT/CCによる注意喚起） [10]
    *   **概要:** 脆弱性を悪用したコンテンツを開いた場合、任意のコードが実行される可能性があります [10]。
    *   **推奨対応:** 最新の Continuous（26.001.21431）または Classic 2024（24.001.30365）バージョンへ更新してください [11]。
*   **Cisco ASAおよびFTDにおける複数の脆弱性**
    *   **CVE番号:** CVE-2025-20333、CVE-2025-20362 [12]
    *   **概要:** 認証なしでのURLエンドポイントへのアクセスと、認証後の任意コード実行を組み合わせた攻撃（ArcaneDoorキャンペーン）が確認されています [12], [13]。
    *   **推奨対応:** Ciscoが提供する最新の修正済みバージョンまたはホットフィックスへのアップデートを速やかに実施してください [14], [15]。

## 🟡 攻撃キャンペーン
*   **クラウドインフラを標的とした「PCPJack」ワームの拡散**
    DockerやKubernetesなどの露出したクラウドサービスをスキャンし、5つの既知の脆弱性を悪用して拡散する新しいマルウェアフレームワーク「PCPJack」が発見されました [16], [17], [18]。このワームは資格情報を窃取するだけでなく、競合する攻撃グループ「TeamPCP」の痕跡を削除してシステムを独占しようとする特徴があります [16], [19]。
*   **ブラジルを拠点とする「TCLBanker」トロイの木馬**
    Logitech AI Prompt Builderのインストーラーに偽装して感染し、59の銀行・暗号通貨プラットフォームを標的にする「TCLBanker」が確認されました [20], [21]。WhatsApp WebやOutlookの連絡先を介して自動的にスパムを送信し、自己拡散する機能を備えています [20], [22], [23]。

## 🤖 AIセキュリティ
*   **AIによるフィッシングの高度化:** ハッカーが生成AIを使用して、現在のフィルタリングを回避する検知困難な「AI Phish」を作成している状況が指摘されています [24], [25]。
*   **インストーラーへの悪用:** マルウェア「TCLBanker」は、LogitechのAI関連ツール（AI Prompt Builder）を装ったトロイの木馬化されたインストーラーを利用して感染を広めています [20]。
*   **AGI（汎用人工知能）のリスクに関する議論:** Elon Musk氏とOpenAIの経営陣との裁判において、超知能AIが人類に及ぼす「破壊的リスク」や倫理的問題が法廷で議論されています [26], [27], [28]。

## 🔵 業界動向
*   **CISA KEVカタログへの追加:** 米CISAは、Ivanti EPMMの脆弱性（CVE-2026-6973）が野生で悪用されていることを受け、既知の悪用された脆弱性カタログ（KEV）に追加しました [29], [30]。
*   **AI安全性に関する対立の表面化:** OpenAIの非営利目的と利益追求のバランスをめぐる裁判が、今後のAI開発の透明性や規制のトレンドに影響を与える可能性があります [27], [31]。
*   **パブリック・プライベート・パートナーシップの重要性:** 民間セクターとの迅速かつ深い連携が、国家のレジリエンス（回復力）に不可欠であるとの見解が示されています [32]。

## 🇯🇵 国内情報
*   **JPCERT/CCによる注意喚起の継続:** Ciscoの脆弱性に関し、修正バージョンの公開前に侵害が発生していた可能性や、国内でも依然として影響を受けるホストが多数存在していることが報告されています [14]。
*   **4月のセキュリティ更新情報の整理:** MicrosoftおよびAdobe製品に関する4月分のセキュリティ更新プログラムの注意喚起が改めて整理され、国内ユーザーへの対策適用が促されています [33], [34]。
