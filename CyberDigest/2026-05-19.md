# 🛡️ サイバーセキュリティ・ダイジェスト 2026-05-19

## 🔴 重大インシデント
- **7-Elevenにおける大規模データ漏洩の確認**：ハッカー集団「ShinyHunters」が、Salesforceから60万件以上の個人情報や企業データを含むレコードを盗み出したと主張し、7-Eleven側もシステムへの侵害を正式に認めました [1-3]。犯行グループは身代金を要求し、応じない場合はデータを売却または公開すると脅迫しています [3]。
- **米国ヘルスケア機関における数百万人の個人情報漏洩**：NYC Health and Hospitals (180万人) やNacogdoches Memorial Hospital (250万人) など、複数の医療機関で大規模なデータ侵害が報告されました [4-6]。流出した情報には、社会保障番号 (SSN) や生体情報、詳細な医療情報が含まれています [4, 5, 7]。
- **Instructureがハッカー集団と身代金支払いで合意**：学校情報ポータル「Canvas」の開発元であるInstructureが、システム侵害を受けた際にShinyHuntersと交渉し、データの破壊と引き換えに身代金を支払う合意に至ったことが報じられました [8]。

## 🟠 脆弱性情報
- **On-Premise Microsoft Exchange Serverの脆弱性 (CVE-2026-42897)** / CVSS 8.1 / 概要：クロスサイトスクリプティング (XSS) の欠陥に起因するスプーフィングの脆弱性で、現在進行形で悪用が確認されています / 推奨対応：Microsoftが提供する暫定的な緩和策の適用 [9]。
- **OpenClawのサンドボックス脱出の脆弱性 (CVE-2026-44112)** / CVSS 9.6 / 概要：AIアシスタントのサンドボックスにおいてレースコンディションを悪用し、サンドボックス外へのデータ書き込みやバックドアの設置を可能にします / 推奨対応：開発者が配布した最新パッチの即時適用 [10, 11]。
- **Burst Statistics WordPressプラグインの脆弱性 (CVE-2026-8181)** / CVSS 9.8 / 概要：未認証の攻撃者が管理者ユーザーになりすまし、サイトを完全に制御できる可能性がある深刻な脆弱性です / 推奨対応：20万以上のサイトが影響を受けるため、最新バージョンへの早急なアップデートが推奨されます [12]。
- **Microsoft 2026年5月セキュリティ更新プログラム** / CVSS 不明（多数） / 概要：リモートでの任意のコード実行などが可能な複数の脆弱性を修正する月例パッチです / 推奨対応：Microsoft Updateを通じた修正プログラムの適用 [13]。

## 🟡 攻撃キャンペーン
- **インターポールによる「Operation Ramz」の展開**：中東および北アフリカ地域において、フィッシングやマルウェア配布、投資詐欺に関与したサイバー犯罪ネットワークが摘発され、201名が逮捕されました [14, 15]。この作戦により53台のサーバーが押収され、約4,000名の被害者が特定されています [15]。
- **Mustang PandaによるAPJ地域への攻撃**：攻撃グループ「Mustang Panda」がDLLサイドローディングを用いて、更新されたツール「FDMTP」を配信し、アジア太平洋地域のホストをプロファイリングする活動が観測されました [12]。
- **TeamPCPによるnpmサプライチェーン攻撃**：Mini Shai-Huludキャンペーンの一環として、TanStackを含む多数のnpmパッケージが汚染され、開発者の資格情報やAPIキーを窃取してクラウドインフラへ侵入する攻撃が拡大しています [16]。

## 🤖 AIセキュリティ
- **AIアシスタント「OpenClaw」の脆弱性チェーン (Claw Chain)**：プロンプトインジェクションや悪意あるプラグインを起点に、4つの脆弱性を連鎖させることでサンドボックスを脱出し、ホストシステムの完全な制御（永続的なバックドアの設置など）が可能です [10, 17, 18]。
- **Hugging Faceでの偽装モデルリポジトリによる攻撃**：OpenAIのプライバシーフィルタ・モデルを偽装し、Windowsユーザーに情報窃取マルウェアを配布する悪意あるリポジトリが確認されました [8]。
- **AIを活用した脆弱性発見と防御の自動化**：OpenAIの「Daybreak」やMicrosoftの「MDASH」など、大量のAIエージェントを用いて自社のコードベースから脆弱性を自動検出し、修正や検証を行う新しい防御の取り組みが発表されています [19]。
