# 🛡️ サイバーセキュリティ・ダイジェスト 2026-05-18

今日取得したソースに基づき、2026年5月18日時点のサイバーセキュリティニュースを以下の形式で分析します。

## 🔴 重大インシデント
*   **GrafanaにおけるGitHubトークン流出とコードベースの窃取および脅迫**
    権限のない第三者が同社のGitHub環境にアクセスできるトークンを入手し、ソースコードをダウンロードしました [1]。攻撃者グループ「CoinbaseCartel」は、盗み出したデータの公開を止めるための身代金を要求しましたが、Grafana側はFBIの助言に従い支払いを拒否しています [2, 3]。
*   **Nvidiaのクラウドゲームパートナーにおけるデータ漏洩**
    アルメニアのパートナー企業GFN.amが侵害を受け、GeForce NOWユーザーの氏名、メールアドレス、電話番号などの個人情報が流出しました [4]。ShinyHuntersを名乗る攻撃者がフォーラムでこのデータベースを10万ドルで販売しようとしたことが確認されています [4]。
*   **Instructure社のCanvasシステムに対する攻撃と和解**
    攻撃者グループShinyHuntersがCanvasシステムへの攻撃を主張し、米国の教育機関でサービス中断が発生しました [5]。同社はこのグループに対し、盗まれたデータの流出を阻止するために支払いを行うという議論を呼ぶ決定を下しています [5, 6]。

## 🟠 脆弱性情報
*   **NGINX PlusおよびOpen Source (CVE-2026-42945)**
    *   **CVSSスコア:** 9.2 [7]
    *   **概要:** `ngx_http_rewrite_module`におけるヒープバッファオーバーフローの脆弱性で、2008年から存在していました [7, 8]。攻撃者は細工したHTTPリクエストにより、ワーカープロセスのクラッシュや、特定の条件下（ASLRが無効な場合）での任意コード実行（RCE）が可能です [9, 10]。
    *   **推奨対応:** F5が提供する最新の修正バージョン（NGINX Plus 37.0.0以降、Open Source 1.31.0以降など）を適用してください [11, 12]。
*   **Microsoft Exchange Server (CVE-2026-42897)**
    *   **CVSSスコア:** 未記載（ゼロデイとして悪用確認） [13]
    *   **概要:** Outlook Web Access (OWA)に影響するスプーフィングおよびクロスサイトスクリプティング (XSS) の脆弱性です [13, 14]。細工されたメールをユーザーがOWAで開くことで、ブラウザコンテキストで任意のJavaScriptが実行される恐れがあります [14]。
    *   **推奨対応:** 恒久的なパッチが公開されるまで、Microsoftが推奨する緩和策（EEMSの有効化など）を実施してください [15, 16]。
*   **openDCIM (CVE-2026-28515, CVE-2026-28517)**
    *   **CVSSスコア:** 9.3 [17]
    *   **概要:** 認可の欠如およびOSコマンド注入の脆弱性です [17]。これらを組み合わせることで、5つのHTTPリクエストのみでリモートからコード実行が可能になります [18]。
    *   **推奨対応:** 最新のアップデートを確認し、脆弱性を修正してください。

## 🟡 攻撃キャンペーン
*   **WooCommerceを標的としたチェックアウトスキミング**
    WordPress用プラグイン「Funnel Builder」の脆弱性を悪用し、チェックアウトページに悪意のあるJavaScriptを注入する攻撃が確認されています [19]。攻撃者はGoogle Tag Managerのスクリプトを装い、顧客のクレジットカード番号や住所を窃取します [20, 21]。
*   **Seedworm（MuddyWater）による韓国メーカー等への攻撃**
    イランに関連するとされる攻撃グループSeedwormが、韓国の主要電子機器メーカーを含む世界中の組織を標的としたキャンペーンを展開しています [22]。署名済みの正当なバイナリを利用したDLLサイドローディングにより、悪意のあるペイロードを配信します [22]。
*   **ArcaneDoorによるCiscoファイアウォールへの持続的な攻撃**
    「UAT-4356」と呼ばれる攻撃者が、Cisco ASAおよびFTDの脆弱性を悪用して「FIRESTARTER」バックドアを設置しています [23]。このマルウェアはログや動作にほとんど痕跡を残さず、パッチ適用後も機器内に潜伏し続ける永続化機能を備えています [23, 24]。

*   **CiscoによるAI駆動型脆弱性評価の仕様公開**
